🚀 Getting Started

Creating a Party

  1. Open Pub Roulette and tap "Create a Party"
  2. Enter your name
  3. Share the 6-character party code with your friends
  4. Customize your crawl settings:
    • Number of Teams: How many teams will compete
    • Number of Pubs: How many locations to visit
    • Search Radius: How far to search for pubs (in meters)
    • Drink Distribution: Random or ensure each selected drink appears once

Joining a Party

  1. Open Pub Roulette and tap "Join a Party"
  2. Enter the party code provided by the host
  3. Enter your name
  4. Wait for the host to start the game

Starting the Game

  1. Host customizes settings (number of pubs, teams, drinks, etc.)
  2. Host taps "Start Game"
  3. App searches for nearby pubs based on location permissions
  4. Players are assigned to teams via the spinning wheel
  5. Drinks are randomly assigned to each pub

⚠️ Common Issues

📍 Location Permission Denied

Problem: "Location permission is required to find nearby pubs"

Solution:

  1. Go to Settings → Privacy → Location Services
  2. Find Pub Roulette and change it to "While Using the App"
  3. Return to the app and try starting the game again

Note: The host needs location permission to search for pubs. If you prefer specific pubs, manually add them before starting.

🍻 Can't Find Pubs

Problem: The app finds no pubs in your area

Solutions:

  • Increase the Search Radius in settings
  • Check that you're in an area with pub/bar listings
  • Try enabling Location Services and allowing "Precise Location" access
  • Manually add custom pubs before starting the game

📷 Camera Permission Denied

Problem: Can't submit photos as proof of visiting pubs

Solution:

  1. Go to Settings → Privacy → Camera
  2. Find Pub Roulette and toggle it On
  3. Return to the app and try submitting a photo

🔄 Team Progress Not Updating

Problem: Photos don't appear after submission, or team progress isn't updating

Solutions:

  • Check your internet connection (WiFi or cellular data required)
  • Make sure all team members have the app open and are on the same party
  • Try closing and reopening the app
  • Ensure everyone has submitted for the current pub before moving to the next

📸 Blurry Photos

Problem: Submitted photos appear with reduced quality

Solution: This is normal. The app compresses photos to reduce upload time and data usage. Photos are verified by your team as proof of location only.

🔧 Troubleshooting

💥 App Crashes When Starting Game

  1. Make sure you're running the latest version of Pub Roulette
  2. Force close the app and reopen it
  3. Check that you have sufficient storage space
  4. If the problem persists, uninstall and reinstall the app

🚫 Can't Connect to Party

Problem: Getting "Party not found" error

Solution:

  • Double-check the party code (it's case-sensitive, 6 characters)
  • Make sure the party hasn't been abandoned
  • Verify you have an active internet connection
  • Try waiting a few seconds and attempting again

🎡 Wheel Spin Doesn't Work

Problem: The team assignment wheel won't spin or freezes

Solution:

  1. Make sure all players are in the party
  2. Check internet connection
  3. Close the app and reopen
  4. Try on a different WiFi network if available

⚡ Choppy or Slow Animations

Problem: Spinning wheel, slot machine, or other animations lag

Solutions:

  • Close other apps to free up device memory
  • Reduce visual effects by checking device settings
  • Ensure you're using a modern iPhone (iPhone 11 or later recommended)
  • Check that your device has sufficient free storage space

📱 Technical Requirements

  • iOS 17.0 or later
  • Internet Required for real-time multiplayer sync
  • Location Required to search for nearby pubs (optional if manually adding pubs)
  • Camera Required to submit photo evidence
  • Storage ~50 MB available

❓ Frequently Asked Questions

Q: Can I play Pub Roulette offline?

A: No, the game requires an active internet connection for real-time multiplayer synchronization.

Q: How long does a party stay active?

A: Parties are stored indefinitely, but players should complete their crawl within a reasonable timeframe. If you need to resume later, use the same party code.

Q: Can I play with more than one team?

A: Yes! When creating a party, specify the number of teams. Players will be assigned to teams via the spinning wheel.

Q: What happens if someone leaves the game?

A: Their team's progress is saved. Other teammates can continue. If the host leaves, another player can take over leadership.

Q: Can I customize which pubs appear in the game?

A: Yes! Before starting the game, you can manually add or remove pubs from the list.

Q: Is there a time limit to complete the crawl?

A: No official time limit, but your team's completion time is tracked on the leaderboard.

Q: Can I play with friends in different cities?

A: Yes! The search location is based on where the host is or a custom location. As long as everyone has an internet connection, you can play together.
